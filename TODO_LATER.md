# Tehtävät, joihin palataan myöhemmin

Tämä tiedosto sisältää tehtäviä, jotka on siirretty myöhemmäksi.

- [ ] **Ratkaise Gemini API -kiintiöongelma:**
    - **Kuvaus:** Gemini API palauttaa 429 "Too Many Requests" -virheen. Tämä on todennäköisesti kiintiöongelma, joka vaatii Google Cloud -projektin asetusten tarkistamista.
    - **Status:** Siirretty myöhemmäksi käyttäjän päätöksellä.

- [ ] **Korjaa checkpoint-komennon toiminta:**
    - **Kuvaus:** `gemini checkpoint` -komento epäonnistui ja tuotti odottamattoman tulosteen. Tämän komennon toiminta on tutkittava ja korjattava.
    - **Status:** Siirretty myöhemmäksi teknisen ongelman vuoksi.
